
## To get started

---
Use Yarn (pref when using expo) --->
`$`  `brew install yarn`  
then `$` ` yarn install`

### Install expo
`$` `yarn add expo-cli -g`


### To run on IOS

`$` `expo start --ios`

### To run on Android 

`$` `expo start --android`

>**if its the first time running it, it may take a while**

to restart the project use `r` as a shortcut to `start`

to clear cache use `-c` added to startup command

e.g: `expo r -c --ios`
